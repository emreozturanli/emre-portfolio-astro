## Project Overview

Bumper is a leading automotive fintech platform that helps drivers "Stay on the road, Pay over time." It allows customers to split the cost of car repairs, MOTs, and servicing into interest-free monthly payments.

I led the frontend development team in building and maintaining the Bumper website, which serves thousands of car dealerships and service partners across the UK, Germany, Spain, and the Netherlands.

### Key Features & Contributions

- **Buy Now Pay Later (BNPL) Core:** Implemented the core flow allowing users to find dealerships, enter details, and get approved for interest-free finance in seconds.
- **Partner Network Integration:** Built features to support a vast network of 5,000+ trusted partners, including franchise dealerships, independent garages, and body shops.
- **Scalable Architecture:** Designed a robust frontend architecture using Next.js and Redux to handle high traffic and complex state management.
- **Internationalization:** Supported multiple regions (UK, DE, ES, NL) with localized content and logic.
- **Performance:** Optimized the application for speed and SEO, ensuring a "Quick fix" experience for users in stressful situations.

### Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **State Management:** Redux Toolkit, RTK Query
- **Styling:** Tailwind CSS
- **Testing:** Jest, Cypress
