## Project Overview

Upay is the core admin dashboard for **UPayments**, a comprehensive online payment solution designed to help businesses grow. It provides a unified interface for merchants to manage their entire financial operation, from invoicing to payment collection.

I played a pivotal role in developing this dashboard, ensuring a seamless and secure experience for business owners.

### Key Features & Contributions

- **Unified Dashboard:** Developed a single-pane-of-glass interface where merchants can track real-time transactions, view deposits, and manage all services.
- **Online Invoicing:** Built a hassle-free invoicing system allowing merchants to create, share, and track digital invoices.
- **Security & Auth:** Implemented robust authentication and authorization flows, ensuring compliance with security standards (PCI DSS Level 1).
- **Data Management:** Leveraged React Query for efficient data fetching and caching, providing real-time updates on sales and revenue.
- **Omni-channel Support:** Ensured the dashboard is responsive and accessible, allowing merchants to "Run Your Business Anywhere."

### Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **State Management:** Redux, React Query
- **Styling:** Tailwind CSS
